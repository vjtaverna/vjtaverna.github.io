What is procedural design?

Procedural design stands against traditional CAD software. Whereas with software like Fusion360, SolidWorks, and Onshape, you design individual pieces step by step, procedural design embraces the idea of creating workflows that can be generalized to all sorts of designs. That's to say, if I create a workflow to make a lattice out of a part, I can simply change the input to another part, and get a new lattice out of it. Oftentimes, these workflows involve random number generation. Once more, I can change the input parameter—in this case the seed for the random number generation—and I can completely change the final design. Check out some of the case studies below for more information.

nTop Latticing

High level procedural CAD software allow you to modify the grid that a lattice structure conforms to. These grid structures can be defined using randomly generated points or using FEA to lightweight parts.
